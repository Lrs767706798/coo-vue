# coo-vue
无疑重复造轮子，只在整理一些平时项目中写过的常用组件。尽量支持各个PC、移动端
.
## Getting Started

```bash
# install
npm install
# run
npm start
```

访问 http://localhost:9000/#/ 开始开发

## Project
```bash

   ├──build/                     * webpack文件
   |
   ├──config/                    * cdn、图片资源等配置
   |
   ├──src/                       * 源码目录
   │   │
   │   │──assets                 * 全局样式、图片资源、字体资源等等
   │   │
   │   └──components             * 各个组件
   │   │   │
   │   │   └──***                * 组件目录
   │   │
   │   └──examples               * 各个组件对应的栗子
   │   │
   │   └──router                 * 路由配置
   │
   │──package.json               * 包信息
   │
   │──.eslintrc.js               * Eslint配置
   │
   │──.babelrc                   * Babel配置
   │
   │──.gitignore                 * Git忽略文件配置
   │
   │──postcss.config.js          * postcss配置文件
```
